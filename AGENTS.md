# AGENTS.md

このリポジトリで作業するすべての AI エージェント（ChatGPT / Codex / Claude Code 等）への指示は **[CLAUDE.md](CLAUDE.md)** に集約している。作業前に必ず読むこと。

要点のみ再掲:

1. 文章は日本語で書く
2. 資格・試験情報は公式確認済みでない限り「要公式確認」と明記し、断定しない
3. 空ファイルを作らない。大きな方針変更は ADR にする
4. ファイル配置・ステータス表記・リンクのルールは [CLAUDE.md](CLAUDE.md) に従う
5. 受験判断・キャリア判断など所有者の意思決定を勝手に確定させない

規約を変更する場合は CLAUDE.md 側を更新し、このファイルは参照のまま維持する。
