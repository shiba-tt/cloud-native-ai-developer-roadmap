# Cloud Native AI Developer Role Definition

## 役割

ユーザー課題を起点に、モバイル / Webアプリ、クラウドバックエンド、AI機能、データ、セキュリティ、運用を統合して設計・実装する開発者です。

## 必要な責任範囲

- アプリケーション設計と実装
- Azureを中心としたクラウド設計
- データストア選定とデータモデリング
- AI / RAG / Agent機能の組み込み
- 認証・認可・シークレット管理
- CI/CD、IaC、監視、運用改善

## できるべきこと

- 要件からアーキテクチャを説明する
- MVPを設計し、小さくリリースする
- コスト、セキュリティ、運用性を考慮する
- AI機能の品質を評価し改善する

## やらないこと

- 資格取得だけを目的化する
- 公式確認なしに試験情報を断定する
- セキュリティやコストを後回しにする
- AIエージェントの出力を無検証で採用する

## iOSエンジニアからの拡張ポイント

- SwiftUIのUX設計力をクラウドAI機能へ接続する
- モバイル認証、通知、オフライン体験をバックエンド設計と統合する
- クライアント視点でAPIとデータ設計を改善する

## 将来的なキャリア候補

- Cloud Native Application Developer
- AI Application Engineer
- Mobile + Cloud Architect
- Developer Productivity / DevOps Engineer
- 個人開発プロダクトのオーナー開発者
