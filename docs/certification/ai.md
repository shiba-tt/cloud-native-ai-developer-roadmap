# AI 系資格

AI 系資格の候補と検討状況。AI Application Phase（[roadmap.md](../roadmap.md) Phase 5）に対応。

> **最重要注意**: AI 系資格は Microsoft の中でも**最も変更が激しい領域**。試験の新設・改編・名称変更・廃止が頻繁に起こる。
>
> - ブログ記事・書籍・AI アシスタントの回答など、**公式ページ以外の情報は一切信用しない**
> - 受験計画に組み込む前に、**必ず Microsoft Learn の公式試験ページ**で、その試験が現在存在するか / 名称 / 言語 / 廃止日 / 後継資格 / 更新日 / 試験範囲を確認する
> - このファイルに書かれた試験番号自体も「候補メモ」にすぎず、現行の正式なラインナップは公式で確認する

## 方針

- AI 系資格は「日本語情報・日本語受験対応が安定してから」取り組む（リポジトリ全体の前提）
- それまでの間も、**資格とは独立して** RAG / Agent / Evaluation の実装スキルは Phase 5 で習得する
- 受験タイミングの判断材料: 日本語受験可否、日本語教材の充実度、試験内容の安定期間（直近の大幅改編からの経過）

## 検討メモ

- 目的は「Azure 上で AI アプリ（RAG / Agent）を実装・運用できること」であり、資格はその体系的な確認手段
- Fundamentals 級は AZ-204 取得済みの現在地から見て学習コストに見合うかを評価する
- Azure AI サービス自体（Azure AI Foundry / Azure OpenAI など）も名称・構成の変更が多いため、学習時は常に公式ドキュメントを一次情報とする
