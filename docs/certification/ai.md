# AI Certifications

AI系資格は変更が激しいため、ブログや古い情報を信用せず、必ず公式 Microsoft Learn の試験ページで現在の状況を確認します。日本語受験、日本語教材、廃止日、後継資格、試験範囲の確認を必須とします。

| 資格 | 位置づけ | 注意点 | 成果物候補 |
|---|---|---|---|
| AI-901 | AI基礎候補 | 存在、言語、廃止日を要公式確認 | AIサービス比較メモ |
| AI-102 | Azure AI開発候補 | 現行試験か要公式確認 | RAG / Azure AIアプリ |
| AI-103 | 要公式確認 | 現行か廃止済みかを確認 | 公式確認後に判断 |
| AI-200 | 要公式確認 | 現行か廃止済みかを確認 | 公式確認後に判断 |

## 判断方針

- AI資格は焦って受験しない
- 日本語情報と公式ページが安定してから検討する
- 受験より先に、小さなAIアプリを作って課題を把握する
