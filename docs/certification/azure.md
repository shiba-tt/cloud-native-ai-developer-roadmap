- 当初は「有効期限の 6 か月前（2026-08 下旬）から更新アセスメントを受ける」計画だったが、**廃止情報が正しい場合、更新アセスメントは廃止日（2026-07-31）で終了し、更新可能期間が始まる前に更新手段がなくなる**。「8 月に更新」を前提にしてはならない

- **最優先タスク**: 所有者が Microsoft Learn の公式ページ（認定ページ / renew ページ / Microsoft Learn プロファイル）で以下を確認する
  - [ ] AZ-204 / Azure Developer Associate の廃止日
  - [ ] 廃止日前に更新アセスメントを受験できるか（更新可能期間の繰り上げ有無）
  - [ ] 取得済み認定が有効期限まで有効か
  - [ ] 後継資格（AI-200 想定）と移行ガイダンス
