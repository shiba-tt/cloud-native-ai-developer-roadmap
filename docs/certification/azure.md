# Azure 系資格（AZ-*）

Azure の開発・管理・設計系資格の候補と検討状況。

> **重要**: 以下の情報はすべて「要公式確認」。受験を決める前に、必ず Microsoft Learn の公式試験ページで試験名・言語・廃止日・後継資格・更新日・試験範囲を確認すること。

## 各資格のメモ

### AZ-204: Developer Associate — ✅ 取得済み

- 本ロードマップの Foundation Phase の土台
- 取得済みだが、知識を整理し成果物に落とし込む作業が残っている（[roadmap.md](../roadmap.md) Phase 1）
- Microsoft 認定資格は年次更新（無料のオンライン更新アセスメント）が必要なはず。**更新期限を公式で確認して記録すること**

### AZ-305: Solutions Architect Expert — 🔍 検討中

- システム設計力の証明として Architecture & DevOps Phase で検討
- Expert レベルは前提となる Associate 資格の要件がある場合があるため、**取得条件を公式で確認**
- 評価軸メモ: 開発者実用性は中〜高（設計視点）、個人開発への応用は高（アーキテクチャ判断）

### AZ-400: DevOps Engineer Expert — 🔍 検討中

- CI/CD / IaC / 監視など、Cloud Native Phase・DevOps 領域と直結
- 前提資格の要件を**公式で確認**
- 成果物候補: CI/CD と監視を含むデリバリーパイプライン（[portfolio-roadmap.md](../projects/portfolio-roadmap.md)）

### AZ-104: Administrator Associate — 🔍 検討中

- 管理者寄りの内容のため、「開発者としての実用性」の評価軸で慎重に判断
- AZ-305 の学習の土台として有効かどうかという観点で評価する
