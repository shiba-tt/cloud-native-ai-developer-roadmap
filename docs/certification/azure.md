# Azure Certifications

Azure系資格は、クラウドアプリケーション開発、アーキテクチャ、DevOpsの基礎を固めるために扱います。すべて受験前に公式 Microsoft Learn の試験ページで要公式確認です。

| 資格 | 状態 | 確認事項 | 成果物候補 |
|---|---|---|---|
| AZ-204 | 取得済み | 更新条件、試験範囲の変更 | Azure Functions / Storage / Messagingを使うAPI |
| AZ-305 | 候補 | 言語、廃止日、後継資格、更新日、試験範囲 | 個人開発プロダクトのアーキテクチャ設計書 |
| AZ-400 | 候補 | 言語、廃止日、後継資格、更新日、試験範囲 | CI/CDと監視を含むデリバリーパイプライン |
| AZ-104 | 候補 | 言語、廃止日、後継資格、更新日、試験範囲 | Azure運用・監視・権限管理の検証環境 |
