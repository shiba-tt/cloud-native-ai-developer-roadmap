# CNCF Certifications

CNCF系資格は、Kubernetesとクラウドネイティブ運用の理解を深めるために検討します。公式ページで試験範囲、言語、更新条件を確認します。

| 資格 | 位置づけ | 成果物候補 |
|---|---|---|
| KCNA | クラウドネイティブ基礎 | Kubernetes基礎メモと用語集 |
| CKAD | Kubernetesアプリ開発 | APIをKubernetesへデプロイ |
| CKA | Kubernetes管理 | クラスタ運用手順の整理 |
| CKS | Kubernetesセキュリティ | セキュリティ設定の検証 |
