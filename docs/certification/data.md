# Data Certifications

データ系資格は、AIアプリケーションに必要なデータ設計、検索、分析基盤を学ぶために検討します。存在、言語、廃止日、後継資格、更新日、試験範囲は公式ページで要確認です。

| 資格 | 位置づけ | 成果物候補 |
|---|---|---|
| DP-420 | Cosmos DB / NoSQL設計候補 | Cosmos DBを使ったモバイルバックエンド |
| DP-300 | SQL / Database運用候補 | SQL設計とバックアップ検証 |
| DP-600 | Fabric / 分析基盤候補 | 個人データ分析ダッシュボード |
| DP-700 | 要公式確認 | データパイプライン検証 |
| DP-800 | 要公式確認 | 試験の存在と内容を確認後に判断 |
