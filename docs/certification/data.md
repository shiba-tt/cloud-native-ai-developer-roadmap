# データ系資格（DP-*）

データプラットフォーム系資格の候補と検討状況。Data Platform Phase（[roadmap.md](../roadmap.md) Phase 2）に対応。

> **重要**: DP 系は改編が多い領域。以下はすべて「要公式確認」であり、**存在の有無・試験言語・廃止日を Microsoft Learn の公式ページで確認するまで、受験計画に組み込まない**こと。廃止済み・後継への統合が行われている可能性が常にある。

## 検討メモ

- 本ロードマップの目的（アプリ開発者としてのデータ設計力）に最も近いのは **Cosmos DB 開発者向け資格（DP-420 想定）**
- 分析系（Fabric 等）はゴールとの接続性が相対的に低いため、評価軸「AI アプリケーション開発との接続性」で慎重に判断する
- 資格に関係なく、SQL / NoSQL / データモデリング / Vector Search の学習自体は Phase 2 で必ず行う
