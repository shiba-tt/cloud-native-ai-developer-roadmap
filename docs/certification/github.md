# GitHub Certifications

GitHub系資格は、日常開発、CI/CD、セキュリティ、AI支援開発の品質を高めるために検討します。公式ページで試験名、範囲、言語、更新条件を確認します。

| 資格 | 位置づけ | 成果物候補 |
|---|---|---|
| GitHub Foundations | GitHub基礎 | Issue / PR / Projects運用の整備 |
| GitHub Actions | CI/CD | テストとデプロイのワークフロー |
| GitHub Copilot | AI支援開発 | AIエージェント利用方針の改善 |
| GitHub Advanced Security | セキュリティ | CodeQLやSecret Scanningの検証 |
