# GitHub 認定資格

GitHub Certifications の候補と検討状況。DevOps 領域（[roadmap.md](../roadmap.md) Phase 4・6）に対応。

> **注**: 受験前に GitHub 公式の認定資格ページ（GitHub Certifications）で、試験の現行ラインナップ・言語（日本語対応の有無）・受験形式・費用を確認すること。

## 候補一覧

| 資格 | 領域 | ステータス | 備考 |
| --- | --- | --- | --- |
| GitHub Foundations | GitHub の基礎 | 🔍 検討中 | 実務経験があるため、学習コストと取得価値のバランスで判断 |
| GitHub Actions | CI/CD / ワークフロー | 🔍 検討中 | **本ロードマップとの親和性が高い候補**。CI/CD 学習の体系化に有効 |
| GitHub Copilot | AI ペアプログラミング | 🔍 検討中 | AI 活用の体系化。試験内容の実用性を評価 |
| GitHub Advanced Security | セキュリティ（コードスキャン等） | 🔍 検討中 | Security Phase との接続。GHAS を使える環境があるかも考慮 |

## 検討メモ

- 本ロードマップでは CI/CD を GitHub Actions で構築する方針のため、**GitHub Actions 認定が最優先候補**
- GitHub 認定は比較的新しい制度のため、日本語受験の可否・試験範囲の安定性を要公式確認
- このリポジトリ自体の運用（Issue / PR / Actions）が、そのまま学習のハンズオンになる

## 成果物との紐付け

- GitHub Actions → このリポジトリおよび個人開発プロダクトの CI/CD パイプライン（[portfolio-roadmap.md](../projects/portfolio-roadmap.md)）
