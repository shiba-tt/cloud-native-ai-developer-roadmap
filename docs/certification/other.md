# その他の資格候補

Azure / CNCF / GitHub / HashiCorp 以外の資格候補を管理する。
現時点では優先度が低いが、キャリア状況の変化に応じて検討する。

> **注**: いずれも受験前に各公式サイトで最新情報（言語 / 形式 / 有効期限 / 費用）を確認すること。

## 方針

- **主軸は Azure**。マルチクラウド資格の追加は「広く浅く」になりやすいため、[principles.md](../principles.md) の原則 6 に照らして慎重に判断する
- 国家資格（IPA 系）は日本語で受験でき情報も安定しているが、実装力への直結度を評価軸で確認する
- ここに載っていない資格を提案されたときも、まず [overview.md](overview.md) の評価軸に通す
