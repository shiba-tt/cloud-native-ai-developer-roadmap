# Certification Overview

資格は学習の方向性を整理し、実装力とキャリア説明力を高めるための手段です。受験前に必ず公式ページで試験名、言語、廃止日、後継資格、更新日、試験範囲を確認します。

## 評価軸

- 開発者としての実用性
- 個人開発への応用可能性
- AIアプリケーション開発との接続性
- 公式情報の安定性
- 日本語教材・日本語受験の有無
- 廃止予定・後継資格の有無
- 学習コスト
- ポートフォリオ化しやすさ

## 運用ルール

1. 候補資格をIssue化する
2. 公式URLを記録する
3. 言語、廃止日、後継資格、更新日、試験範囲を確認する
4. 合格後に作る成果物を先に決める
5. 学習ログと成果物を紐付ける
