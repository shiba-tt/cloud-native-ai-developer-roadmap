# Security Certifications

セキュリティ系資格は、認証・認可、ID管理、監視、クラウドセキュリティ設計を学ぶために検討します。存在、言語、廃止日、後継資格、更新日、試験範囲は公式ページで要確認です。

| 資格 | 位置づけ | 成果物候補 |
|---|---|---|
| SC-300 | Identity / Entra ID候補 | OAuth / OIDC / Entra ID認証サンプル |
| SC-100 | セキュリティアーキテクチャ候補 | ゼロトラスト観点の設計レビュー |
| SC-200 | セキュリティ運用候補 | ログ監視とインシデント対応メモ |
| SC-500 | 要公式確認 | 試験の存在と内容を確認後に判断 |
