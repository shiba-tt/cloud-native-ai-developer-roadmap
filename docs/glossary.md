# Glossary

## Cloud Native

コンテナ、マネージドサービス、IaC、CI/CD、監視などを活用し、変更に強く運用しやすいシステムを作る考え方。

## RAG

Retrieval-Augmented Generationの略。外部データ検索と生成AIを組み合わせ、根拠に基づいた回答を生成する構成。

## Agent

目的達成のために、モデルがツール呼び出し、検索、計画、実行を組み合わせて動作するアプリケーションパターン。

## IaC

Infrastructure as Codeの略。クラウドリソースをコードで定義し、再現性とレビュー可能性を高める手法。

## Observability

ログ、メトリクス、トレースを使ってシステムの状態を理解し、問題発見と改善につなげる考え方。
