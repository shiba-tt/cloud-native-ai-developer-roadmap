# Courses — オンラインコース / ラーニングパス

オンラインコースと Microsoft Learn ラーニングパスの管理。

## 方針

- Microsoft / Azure 系は **Microsoft Learn のラーニングパス**を第一候補にする（無料・公式・最新）
- 資格対応のラーニングパスは、公式試験ページの「Study guide」からたどる
- 有料コース（Udemy 等）は、ハンズオンの流れを掴む用途に限定し、公開日・更新日を確認してから購入する
