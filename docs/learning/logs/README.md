# Study Logs — 学習ログの保存先

[templates/study-log.md](../../../templates/study-log.md) をコピーして記入した学習ログを、このディレクトリに保存する。

## 命名規則

```text
docs/learning/logs/YYYY/YYYY-MM-DD.md

例:
docs/learning/logs/2026/2026-07-15.md
```

年ごとのサブディレクトリ（`2026/` など）は最初のログを書くときに作成する。

## 運用ルール

- 完璧に書くより続けることを優先する。数行でもよい
- 軽い記録は関連 Issue のコメントに書くだけでもよい。**週単位以上のまとまった学びはこのディレクトリにファイルとして残す**
- 月次レビュー時に当月のログを見返し、成果と詰まりを抽出する
