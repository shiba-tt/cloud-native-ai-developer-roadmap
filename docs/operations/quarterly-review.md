# Quarterly Review — 四半期レビュー記録

四半期ごとに、月次レビューより一段高い視点でロードマップ全体を見直す。

## 運用ルール

- 3 か月ごとに Issue（review テンプレート）を起票して実施する
- 記録はこのファイルに新しい四半期を**上に**追記していく
- フェーズの重心移動・資格計画の変更など、大きな方針転換は ADR に記録する

## レビュー観点

- **フェーズ進捗**: [roadmap.md](../roadmap.md) の現在フェーズは適切か。重心を移すべきか
- **資格計画**: 検討中の資格の公式情報を再確認したか（特に AI 系は 3 か月で状況が変わり得る）
- **成果物**: [portfolio-roadmap.md](../projects/portfolio-roadmap.md) の進捗。学習が成果物に繋がっているか
- **キャリア**: [career/yearly-goals.md](../../career/yearly-goals.md) とのずれはないか
- **原則**: [principles.md](../principles.md) に反する行動をしていないか（資格が目的化していないか等）
- **コスト**: クラウド費用・教材費・受験料は計画内か
