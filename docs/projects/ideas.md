# Ideas — プロダクトアイデアのストック

個人開発プロダクトのアイデアを貯めておく場所。
思いついたら気軽に追記し、四半期レビューで棚卸しする。

## 評価の観点

- 自分が使いたいか（ドッグフーディングできるか）
- ロードマップの技術（クラウド / データ / AI / セキュリティ）を自然に使えるか
- MVP を 1〜2 か月で出せるサイズに切れるか
- 運用コストが個人開発として現実的か
