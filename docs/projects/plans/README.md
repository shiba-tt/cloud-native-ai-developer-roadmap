# Project Plans — プロジェクト計画の保存先

[templates/project-plan.md](../../../templates/project-plan.md) をコピーして記入したプロジェクト計画を、このディレクトリに保存する。

## 命名規則

```text
docs/projects/plans/<プロジェクト名をケバブケースで>.md

例:
docs/projects/plans/study-log-assistant.md
```

## 運用ルール

- プロジェクト用 Issue（project テンプレート）からこの計画にリンクする
- MVP・完了条件の変更はこのファイルを直接更新する（履歴はコミットに残る）
- 完了・中断後もファイルは削除せず、結果と学びを追記して [portfolio-roadmap.md](../portfolio-roadmap.md) に反映する
