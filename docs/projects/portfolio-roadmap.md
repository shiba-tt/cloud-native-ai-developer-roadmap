# Portfolio Roadmap

| 資格・学習領域 | 成果物 | 説明できること |
|---|---|---|
| DP-420 | Cosmos DBを使ったモバイルバックエンド | NoSQLモデリング、パーティション設計、API連携 |
| SC-300 | Entra ID / OAuth / OIDCを使った認証サンプル | 認証フロー、認可、トークン管理 |
| CKAD | Kubernetes上にAPIをデプロイ | コンテナ、Pod、Service、ConfigMap、Secret |
| Terraform | AzureリソースをIaC化 | 再現性、レビュー可能性、状態管理 |
| AI系 | RAG / Agentアプリ | 検索、プロンプト、評価、ガードレール |
| AZ-400 | CI/CDと監視を含むデリバリーパイプライン | 自動テスト、デプロイ、監視、改善 |
| AZ-305 | 個人開発プロダクトの設計書 | 可用性、拡張性、コスト、セキュリティ |

## 方針

資格ごとに「合格した」で終わらせず、必ず成果物、設計メモ、振り返りを残します。
