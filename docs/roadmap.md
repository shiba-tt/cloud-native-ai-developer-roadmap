# Roadmap

## 1. Foundation Phase

- AZ-204で得たAzure開発基盤を整理する
- App Service、Functions、Storage、Messaging、Monitoringの基本を復習する
- クラウドアプリケーション開発の基礎を固める

## 2. Data Platform Phase

- SQL / NoSQL / Cosmos DB / データモデリングを学ぶ
- モバイルバックエンドに必要なデータ設計を実践する
- DP系資格を検討する

## 3. Security & Identity Phase

- Entra IDの基礎を学ぶ
- OAuth / OIDCのフローを理解する
- Managed IdentityとKey Vaultを実装する
- SC系資格を検討する

## 4. Cloud Native Phase

- Dockerでアプリケーションをコンテナ化する
- KubernetesにAPIをデプロイする
- TerraformなどでIaC化する
- Observabilityを設計する
- CNCF / HashiCorp系資格を検討する

## 5. AI Application Phase

- Azure AIとAzure OpenAIの公式情報を確認する
- RAG構成を設計・実装する
- AgentパターンとEvaluationを学ぶ
- AI系資格を検討する

## 6. Architecture & DevOps Phase

- AZ-305でアーキテクチャ設計を学ぶ
- AZ-400でDevOpsとデリバリーを学ぶ
- システム設計、CI/CD、運用設計を統合する

## 7. Portfolio Phase

- 学んだ内容を個人開発プロダクトとして形にする
- 設計ドキュメント、運用手順、デモ、振り返りを残す
- キャリア上の実績として説明できる形に整える
