# Vision

## 目指す姿

Cloud Native AI Developer は、ユーザー課題を起点に、モバイルアプリ、Webアプリ、クラウドバックエンド、AI機能、データ基盤、運用までを一貫して設計・実装できる開発者です。

## 作れるようになりたいもの

- Swift / SwiftUI を活かしたAI対応モバイルアプリ
- Azure Functions、App Service、Container Apps、AKSなどを使ったバックエンド
- SQL / NoSQL / Vector Search を組み合わせたデータ基盤
- Azure OpenAIやAzure AI Servicesを活用したRAG / Agentアプリケーション
- 認証・認可、監視、CI/CD、IaCを備えた運用可能なプロダクト

## 扱う技術領域

- Application Development
- Azure Platform
- Data Platform
- AI / RAG / Agent
- Security / Identity
- Cloud Native / Kubernetes
- DevOps / IaC / CI/CD
- Observability
- Product Development
- Mobile / iOS

## なぜクラウドネイティブとAIを組み合わせるのか

AI機能は単体では価値になりにくく、アプリケーション、データ、認証、運用、コスト管理と結びついて初めて継続的な価値になります。クラウドネイティブな設計を身につけることで、AIを安全かつ継続的に改善できるプロダクトへ組み込めます。

## iOSエンジニアとしての強み

- ユーザー体験から逆算して機能を設計できる
- モバイル特有の制約、認証、通知、オフライン対応を理解している
- Swift / SwiftUI によるクライアント実装力を、AIバックエンドやクラウド設計と接続できる

## 5年後の理想像

5年後には、モバイルアプリからクラウドバックエンド、AI機能、データ設計、運用までを自律的に設計し、個人開発プロダクトとして継続改善できる状態を目指します。資格はその過程を確認するマイルストーンとして扱います。
